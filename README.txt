####-------------------------------------Fuction--------------------------------------####
开关机+换档+RGB+RW+PreHeat+0.8Ω

PCB板				主板+USB板
LED灯色				主板RGB
输出电压			三挡输出:一档2.7V(红),二档3.2V(蓝灯),三挡3.7V(绿灯)
咪头灵敏度			-100Pa~-200Pa
静态电流			≤10UA

上电				上电后主板红白绿各闪1次，然后进入待机模式.上电默认关机。
休眠				无任何操作3S后进入休眠。
正常吸烟			咪头/按键启动吸烟，吸烟时当前档位灯1S渐亮，停止吸烟1S渐灭。
开/关机				开机状态下，2S内连续按键5次，当前档位灯闪5次，进入关机状态；
					关机状态下，2S内连续按键5次，当前档位灯闪5次，进入开机状态；
					关机状态下吸烟咪头不触发吸烟。

预热				开机状态下,2S内按按键2次,进入预热状态,当前档位灯呼吸渐亮渐灭,输出电压1.8V,持续时间10S(2S一次呼吸)
档位				开机状态下,2S内按按键3次,切换下一个档位,低中高三个档位循环切换,上电默认低档. 切换到对应档位,对应档位灯闪3次.(低档红灯,中档蓝灯,高档绿灯)

低电保护			吸烟前电池电压小于3.2V±0.1V或吸烟过程中电池电压小于3.0V±0.1V时，当前档位灯闪烁10次提示，停止输出，进入待机，连续低电报警3次后进入关机状态，快按5次按键或插充电器开机。
短路/低阻值保护		当输出负载小于0.8Ω±0.1以下或检测雾化短路时，吸烟时当前档位灯长亮2S后熄灭，雾化器停止输出
超时保护			单次吸烟超过10秒（±1秒）时，当前档位灯灯闪3次提示超时，停止输出。
ESD保护				TVS保护,直接接触4KV,间接8KV

充电保护			OVP保护,最大充电耐压24V
充电电流			5V/500mA±50mA。
充电指示			充电时插入当前档位闪3次后常亮、充满灭灯 、拔掉充电线,当前档位灯闪3次
充满电压			4.2V（±0.05V）,充满复充电压4.05V
边充电边吸烟		支持。
0V充				支持。
